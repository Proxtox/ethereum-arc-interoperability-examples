# ethereum-arc-interoperability-examples

Educational examples of cross-chain messaging and data sharing between Ethereum and Arc.

## Overview
Starter repository demonstrating patterns for interoperability between Ethereum mainnet and Circle's Arc L1 (stablecoin-native chain).

## Tech Stack
- Solidity
- CCTP / LayerZero / Axelar (for production)
- viem

## Getting Started
```bash
git clone https://github.com/Proxtox/ethereum-arc-interoperability-examples.git
cd ethereum-arc-interoperability-examples
npm install
```

## Key Files
- `contracts/CrossChainMessenger.sol` - Basic send/receive message example
